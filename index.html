select by alexyoung

A JavaScript database library

Sick of ORM and schemas? select gets right to the point!

Supported databases:

  • MySQL
  • MongoDB
  • Memory
  • Planned: PostgreSQL, SQLite, CouchDB
  • Researching: Riak, Redis, selector language

Dependencies

MongoDB:

  • github.com/christkv/node-mongodb-native
  • npm: mongodb-native

MySQL:

  • github.com/felixge/node-mysql
  • npm: mysql

Testing:

  • github.com/felixge/node-mysql
  • npm: whiskey

Install

npm install select

Examples

  // Find some items and iterate
  select('users')
    .find({ name: 'Alex' })
    .limit(3)
    .offset(8)
    .each(function(index) {
      console.log(this);
    });

  // Update an attribute
  select('users')
    .find(17)
    .attr({ name: 'Yuka' });

  // Create
  select('users')
    .add({
      name: 'Bob',
      email: 'bob@example.com'
    });

  // Delete
  select('users')
    .find(1)
    .del();

  // Selector language
  select('users[name="Alex", email="alex@example.com"]').
    values(function(values) {});
